FTPClient is a very simple FTP, FTPS and SFTP client for Android 5.0+, using Apache Commons Net (https://commons.apache.org/proper/commons-net) and SSHJ (https://github.com/hierynomus/sshj/).

<b>Features</b>

- view directory content
- upload files to servers
- create directories
- download files
- view file and directory properties
- rename files and directories
- view text and image files
- edit text files
